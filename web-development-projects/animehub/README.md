# AnimeHub

## Overview

AnimeHub is a full-stack web application project built using Flask and PostgreSQL. The project focuses on creating a dynamic website with back-end logic, database integration, and user-facing pages.

## Purpose

The purpose of this project was to apply full-stack web development concepts by connecting a front-end interface with a Flask back-end and a PostgreSQL database. It helped demonstrate how web applications can store, retrieve, and manage data through a database-driven system.

## Key Features

- User authentication
- Dynamic web pages
- Database integration
- Server-side routing
- Back-end logic using Python and Flask
- Front-end pages using HTML, CSS, and JavaScript
- PostgreSQL database support
