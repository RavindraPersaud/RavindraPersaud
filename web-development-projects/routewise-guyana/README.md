# RouteWise Guyana

## Overview

RouteWise Guyana is an Internet Computing I web application project designed to provide public transportation route and fare information for Guyana. The project focuses on helping users find route details, fare estimates, pickup points, landmarks, and travel guidance through a simple web-based interface.

## Purpose

The purpose of this project was to apply web development concepts from Internet Computing I to a real-world local problem. Public transportation information in Guyana can be difficult for commuters, students, and first-time travelers to access, so this project explored how a website could make route and fare information easier to find.

## Key Features

* Public transportation route information
* Fare details and estimates
* Route directory
* Pickup point and landmark information
* Responsive web pages
* Contact or feedback functionality
* User-facing interface for commuters
